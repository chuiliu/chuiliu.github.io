Git 常用命令详解

G
开发工具
2016-04-06 10:04

备忘，记录一些常用的 git 命令，现在只会用一些最简单的，以后慢慢完善

  • 配置
    // 配置用户信息
git config --global user.name "[name]"
git config --global user.email "[email address]"
  • 常用
// 初始化git仓库
git init

// 查看当前仓库的状态
git status

// 查看提交日志
git log
  • add
// 添加当前目录下所有文件（3种方式）
git add .
git add -A
git add --all
// 添加文件
git add [filename]
// 删除和移动的文件也能提交到暂存区
git add --all
  • commit
// 提交到仓库
git commit -m "descriptions"
git commit [filename] -m "descriptions"
  • 分支
// 查看当前分支
git branch

// 新建分支
git branch [branchName]

// 新建并切换到分支
git checkout -b [branchName]

// 切换到分支
git checkout [branchName]

// 提交分支
git push origin [branchName]

// 合并分支
git merge [branchName]

// 删除分支
git branch -d [branchName]
  • remote
// 连接到远程仓库
git remote add origin git@github.com:chuiliu/test.git

// 推送到远程仓库
git push -u origin master

// 远程仓库更新到本地
git pull git@github.com:chuiliu/test.git [branchName]

// 从远程仓库克隆
git clone git@github.com:chuiliu/test.git
