去除 inline-block 之间的空白间隔

  • CSS
  • 前端

当多个连续的inline-block元素水平方向排列时，各个元素之间会出现空白间隔，这是由于空白字符（空格或换行）引起的。

有时候因为实际布局需要，或者为了避免不必要的影响，必须去除空白，下面总结了几种常用的解决方案：

  • 移除空白字符，全部写在同一行

  • 改变换行的位置

  • 设置父元素的font-size为0，再为每个inline-block对象设置font-size的值
