Sublime Text 常用快捷键与配置项

S
开发工具
2016-01-26 10:01

记录 sublime Text 3 编辑器中一些常用的快捷键

ctrl + d     # 选择相同的单词，连续按继续选择
alt + F3     # 选择整个文档中相同的单词
ctrl + g     # 输入行号，跳到某一行


ctrl + l                 # 选择一行，连续按继续选择
ctrl + shift + ← / →     # 向左/向右选择文本
ctrl + shift + ↑ / ↓     # 向上/向下移动行
ctrl + shift + k         # 删除一行
ctrl + enter             # 在当前行下插入一行
ctrl + shift + enter     # 在当前行上插入一行


ctrl + shift + d     # 向下复制
ctrl + shift + v     # 粘贴，并按当前文本格式化
ctrl + x             # 剪切一行
ctrl + k + k         # ctrl+k后再按一次k，删除从光标处至行末的文本
ctrl + p             # 搜索项目中的文件


alt + .              # 闭合html标签
ctrl + /             # 单行注释
ctrl + shift + /     # 多行注释


ctrl + pagedown      # 切换到左边的标签页
ctrl + pageup        # 切换到右边的标签页
ctrl + w             # 关闭当前文件
ctrl + shift + t     # 恢复上一个关闭的标签
alt + shift + 2      # 分两屏

记录sublime Text编辑器中一些常用的配置项：Preferences > Settings-User

  • 在新窗口中打开文件夹：

    "open_files_in_new_window": true

  • 按tab自动补全：

    "auto_complete_commit_on_tab": true

  • 设置unix换行：

    "default_line_ending": "unix"

  • 把tab转为空格：

    "translate_tabs_to_spaces": true

  • 设置保存文件时自动删除无用的空格：

    "trim_trailing_white_space_on_save": true

  • 将用连接符“-”连接的多个单词识别为一个单词，在双击选中单词的时候很有用，特别是CSS的选择器命名通常使用“-”来连接单词：

    将   "word_separators": "./\\()\"'-:,.;<>~!@#$%^&*|+=[]{}`~?"
改为 "word_separators": "./\\()\"':,.;<>~!@#$%^&*|+=[]{}`~?"
